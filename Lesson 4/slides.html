What Are Functions?

üì¶

Functions are reusable packages of code

üîÑ The DRY Principle

Don't Repeat Yourself - Write once, use many times!

Functions solve the copy-paste problem by letting us package instructions and reuse them everywhere.

The Copy-Paste Trap

üòµ

Look at this repetitive code:

// BAD: Lots of repetition!
playerGold = playerGold + 10;
document.getElementById('gold').innerHTML = playerGold;
document.getElementById('story').innerHTML = "You found 10 gold!";

playerGold = playerGold + 15;
document.getElementById('gold').innerHTML = playerGold;
document.getElementById('story').innerHTML = "You found 15 gold!";

Problem: Same code, different numbers!

The Smart Solution

‚ú®

One function handles everything:

// GOOD: One flexible function
function findGold(amount, location) {
    playerGold = playerGold + amount;
    document.getElementById('gold').innerHTML = playerGold;
    document.getElementById('story').innerHTML =
        "You found " + amount + " gold in the " + location + "!";
}

// Usage - so much cleaner!
findGold(10, "forest");
findGold(15, "cave");

Function Fundamentals

üì¶ Functions are like recipes

Write the instructions once, follow them many times

    document.getElementById('story').innerHTML = "Welcome, adventurer!";
}

// Function call (following the recipe)

Parameters: The Magic Inputs

üéõÔ∏è Parameters make functions flexible

Same function, different inputs = different results

// Parameters are like mail slots
function greetPlayer(playerName) {
    document.getElementById('story').innerHTML =
        "Welcome, brave " + playerName + "!";
}

// Same function, different results!
greetPlayer("Alex");     // "Welcome, brave Alex!"
greetPlayer("Jordan");   // "Welcome, brave Jordan!"

Functions That Modify Game State

üî• Using conditionals from Lesson 3!

let playerHealth = 100;

function changeHealth(amount) {
    playerHealth = playerHealth + amount;

    // Keep health between 0 and 100 using conditionals
    if (playerHealth > 100) playerHealth = 100;
    if (playerHealth < 0) playerHealth = 0;
    // Update the display
    document.getElementById('health').innerHTML = playerHealth;
}
// Use positive numbers to heal:
changeHealth(20);   // Heal 20 points
// Use negative numbers for damage:
changeHealth(-30);  // Take 30 damage

Essential Game Helper Functions

// 1. Update story text
function updateStory(text) {
    document.getElementById('story').innerHTML = text;
}

// 2. Manage inventory
function hasItem(itemName) {
    return inventory.includes(itemName);
}

function addItem(itemName) {
    if (!hasItem(itemName)) {
        inventory.push(itemName);
        updateStory("You found a " + itemName + "!");
    }
}

Return Values

üì§ Functions can give back information

// Functions that answer questions
function canAfford(cost) {
    if (playerGold >= cost) {
        return true;
    } else {
        return false;
    }
}

// Using the returned information
if (canAfford(50)) {
    buyItem("sword");
} else {
    updateStory("You can't afford that!");
}

Smart Game Systems

Combining Functions + Conditionals

// One function handles all doors!
function tryOpenDoor(doorColor, requiredKey, newRoom) {
    if (hasItem(requiredKey)) {
        updateStory("The " + doorColor + " door opens!");
        currentRoom = newRoom;
        return true;
    } else {
        updateStory("You need a " + requiredKey + "!");
        return false;
    }
}

// Usage - no more copy-paste!
tryOpenDoor("red", "Red Key", "treasure_room");
tryOpenDoor("blue", "Blue Key", "library");
