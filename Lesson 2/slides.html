What Are Variables?

üì¶

Think of Variables as Labeled Boxes

  • üì¶ Box Label = Variable name
  • üìù What's Inside = Variable value
  • üîÑ Can Change = We can put new things in the box

JavaScript Variables in Action

// Creating variables (let = "let me create a box")
let playerName = "Alex";
let playerHealth = 100;
let currentLocation = "forest";

// Changing variable values
playerHealth = 75;        // Took some damage!
currentLocation = "cave"; // Moved to a new place

Types of Variables We'll Use

Text (Strings)

let storyText = "Once upon a time...";
let characterName = "Brave Knight";

Always use quotes for text!

Numbers

let goldCoins = 50;
let playerLevel = 1;

No quotes needed for numbers

True/False (Booleans)

let hasKey = false;
let doorIsOpen = true;

Only two values: true or false

Connecting Buttons to Functions

// When button is clicked, run the function
<button onclick="openDoor()">Open door</button>

// This calls (runs) our function:
function openDoor() {
    document.getElementById('story').innerHTML = "Door creaks open!";
}

onclick = "Run this function when clicked"

Changing Web Pages with Code

// Find the element with id="story"
let storyElement = document.getElementById('story');

// Change what's inside it
storyElement.innerHTML = "New story text!";

// Or do it in one line:
document.getElementById('story').innerHTML = "Even newer text!";

This is how we make web pages interactive!
