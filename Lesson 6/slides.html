What Are Objects?

Objects are smart containers that hold both data AND functions!

// Simple object with properties let player = { name: "Alex", health: 100, level: 1 }; // Access properties console.log(player.name); // "Alex" player.health = 75; // Change value

Think of objects as labeled boxes with built-in tools

Objects Beat Scattered Variables

‚ùå BAD: Messy

let playerName = "Alex"; let playerHealth = 100; let playerInventory = ["sword"]; function playerTakeDamage(amount) { playerHealth -= amount; }

‚úÖ GOOD: Organized

let player = { name: "Alex", health: 100, inventory: ["sword"], takeDamage(amount) { this.health -= amount; } };

Objects with Methods

Methods are functions that belong to objects

let character = { name: "Hero", health: 100, takeDamage: function(amount) { this.health -= amount; // 'this' refers to the character if (this.health <= 0) { this.die(); } }, heal: function(amount) { this.health += amount; console.log(`Healed ${amount}! Health: ${this.health}`); } };

The Magic of 'this'

this always refers to the object that owns the method

let wizard = { name: "Gandalf", mana: 100, castSpell: function(cost) { this.mana -= cost; // 'this' = wizard object console.log(`${this.name} has ${this.mana} mana left`); } }; wizard.castSpell(20); // "Gandalf has 80 mana left"

It's like the object talking about itself! ü™Ñ
