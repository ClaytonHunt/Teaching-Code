Arrays: The Smart Solution

üì¶
// GOOD: One array handles unlimited items
let inventory = ["Sword", "Shield", "Potion", "Key"];

// Arrays start counting at 0!
inventory[0]  // "Sword" (first item)
inventory[1]  // "Shield" (second item)
inventory[2]  // "Potion" (third item)

üóÉÔ∏è Arrays are like numbered filing cabinets

Each slot has a number (index) and can hold one piece of data

Essential Array Methods

üõ†Ô∏è
let items = ["Apple", "Bread"];

// Add to end
items.push("Cheese");    // ["Apple", "Bread", "Cheese"]

// Remove from end
let lastItem = items.pop();  // Removes "Cheese"

// Check if item exists
let hasApple = items.includes("Apple");  // true

// Get array size
let count = items.length;    // 2

For Loops: Automation Power

üîÑ
// Without loops (BAD):
console.log("Count: 0");
console.log("Count: 1");
console.log("Count: 2");
// ... copy-paste nightmare!

// With loops (GOOD):
for (let i = 0; i < 3; i++) {
    console.log("Count: " + i);
}
// Prints: 0, 1, 2

One loop replaces endless copy-paste!

Understanding For Loop Parts

üîß
for (let i = 0; i < 5; i++) {
    // Your code here
}

üéØ Breaking it down:

  • let i = 0 - Start counter at 0
  • i < 5 - Keep going while i is less than 5
  • i++ - Add 1 to i each time

Arrays + Loops = Magic

‚ú®
let enemies = ["Goblin", "Orc", "Dragon"];

// Process each enemy automatically
for (let i = 0; i < enemies.length; i++) {
    console.log("Fight the " + enemies[i]);
}

// Simpler version (for...of loop)
for (let enemy of enemies) {
    console.log("Fight the " + enemy);
}

Process any number of items with one loop!

Building Game Systems

üéÆ
// Inventory system with arrays and loops
function countPotions(inventory) {
    let potionCount = 0;
    for (let item of inventory) {
        if (item === "Potion") {
            potionCount++;
        }
    }
    return potionCount;
}

let bag = ["Sword", "Potion", "Potion", "Key"];
console.log(countPotions(bag));  // 2

Smart Shop System

üõçÔ∏è
let shopItems = [
    { name: "Sword", price: 100 },
    { name: "Shield", price: 75 },
    { name: "Potion", price: 25 }
];

// Show affordable items
function showAffordableItems(playerGold) {
    for (let item of shopItems) {
        if (playerGold >= item.price) {
            console.log("You can buy: " + item.name);
        }
    }
}

While Loops: Unknown Repetition

‚ùì
// When you don't know how many times
let health = 100;
let poisoned = true;

while (poisoned && health > 0) {
    health -= 5;
    console.log("Poison damage! Health: " + health);

    if (Math.random() > 0.7) {
        poisoned = false;
        console.log("Poison wore off!");
    }
}

Perfect for game loops and uncertain conditions!
